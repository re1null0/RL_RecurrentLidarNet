# RL_RecurrentLidarNet

This repository contains reinforcement learning agents for the F1TENTH simulator.

## Supported algorithms
- `dqn`
- `ppo`
- `muesli`
- `tdmpc`

Use the configuration files in `rln_skrl/configs` to select the desired algorithm. A sample TD-MPC configuration is available at `rln_skrl/configs/tdmpc.yaml`.
